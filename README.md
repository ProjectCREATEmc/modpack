# Modpack Installation Guide (Prism Launcher)

This guide explains how to install this modpack using **Prism Launcher**.

---

## Requirements

Before installing, make sure you have:

- Latest version of **Prism Launcher**
- A valid **Minecraft Java Edition** account
- At least **6-8 GB of RAM** available (8 GB recommended)

Download Prism Launcher here:
https://prismlauncher.org/

---

# Installation

### Step 1 - Download the Modpack

Download the latest `.zip` release from the GitHub Releases page.

Example:

```
MyModpack-v1.0.zip
```

Do **not** extract the ZIP file.

---

### Step 2 - Open Prism Launcher

Launch **Prism Launcher**.

---

### Step 3 - Add an Instance

Click:

```
Add Instance
```

---

### Step 4 - Import the ZIP

Select:

```
Import
```

Browse to the downloaded modpack ZIP.

Select the file and click:

```
Open
```

Prism Launcher will automatically import everything.

---

### Step 5 - Wait for Installation

Prism Launcher will:

- Download Minecraft
- Download Fabric/Forge/NeoForge (if required)
- Download all mods
- Create the instance

This may take several minutes depending on your internet connection.

---

### Step 6 - Launch

Double-click the new instance.

The first launch may take several minutes while Minecraft initializes.

---

# Allocating More RAM

If the modpack runs slowly or crashes with memory errors:

Right-click the instance and select:

```
Edit
```

Open:

```
Settings
```

Enable:

```
Memory
```

Recommended values:

| System RAM | Allocate |
|------------|----------|
| 8 GB | 4 GB |
| 16 GB | 6 GB |
| 32 GB | 8 GB |

Avoid allocating more than half of your system memory.

---

# Updating the Modpack

When a new version is released:

1. Download the new ZIP.
2. Import it into Prism Launcher.
3. If prompted, choose to update the existing instance.

Always back up your saves before updating.

---

# Troubleshooting

## Missing Java

Install Java 21 (or the version required by the modpack).

Prism Launcher can usually download and manage Java automatically.

---

## Game Crashes on Launch

Try the following:

- Restart Prism Launcher
- Verify your internet connection
- Allocate more RAM
- Update graphics drivers
- Make sure you are using the latest modpack version

---

## Mods Failed to Download

Some antivirus software or firewalls may block downloads.

Try:

- Clicking **Retry**
- Restarting Prism Launcher
- Temporarily disabling aggressive antivirus protection

---

**No.**

Import the ZIP directly into Prism Launcher.

---

You can, but additional mods are **not officially supported** and may cause issues.
